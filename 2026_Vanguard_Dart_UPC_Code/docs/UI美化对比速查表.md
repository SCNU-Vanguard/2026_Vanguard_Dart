## 🔄 升级指南

### 如何应用这些改进

如果你有旧版本的上位机，只需要更新 `main_app.py` 中的 `STYLE_SHEET` 部分（第28-203行）即可。

### 回退方案

如果觉得新样式不适应，可以调整以下参数：

```python
# 降低字体权重
font-weight: 600;  →  font-weight: 500;

# 减小圆角
border-radius: 15px;  →  border-radius: 10px;
border-radius: 8px;   →  border-radius: 5px;

# 减小边框
border: 2px;  →  border: 1px;

# 减小内边距
padding: 15px;  →  padding: 10px;
padding: 8px;   →  padding: 6px;
```

## 🎯 设计原则

本次UI美化遵循以下原则：

1. **清晰性**: 字体加粗，提升可读性
2. **现代感**: 大圆角，符合当代审美
3. **一致性**: 统一的圆角和边框体系
4. **呼吸感**: 增加内边距，避免拥挤
5. **层次感**: 通过边框粗细区分组件
