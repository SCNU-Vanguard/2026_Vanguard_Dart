# UI字体加粗和圆角优化说明

## 📐 优化总览

对上位机助手的UI进行了全面的字体加粗和圆角美化，提升了界面的现代感和可读性。

---

## ✨ 具体改进

### 1. 字体加粗优化

#### 全局字体
- **之前**: `font-weight: normal` (默认)
- **现在**: `font-weight: 600` (半粗体)
- **效果**: 所有文字更清晰、更易读

#### 标签文字 (QLabel)
- **之前**: `font-weight: 500`
- **现在**: `font-weight: 600`
- **效果**: 标签文字更醒目

#### 输入框和下拉框 (QLineEdit, QComboBox)
- **之前**: 默认字重
- **现在**: `font-weight: 600`
- **效果**: 输入内容更清晰

#### 文本框 (QTextEdit)
- **之前**: 默认字重
- **现在**: `font-weight: 500` (等宽字体，稍细以保持可读性)
- **效果**: 日志内容平衡美观和密度

#### 复选框 (QCheckBox)
- **之前**: 默认字重
- **现在**: `font-weight: 600`
- **效果**: 选项文字更清晰

#### 按钮 (QPushButton)
- **之前**: `font-weight: bold`
- **现在**: `font-weight: bold` (保持)
- **效果**: 按钮文字已经很粗，无需修改

#### 标签页 (QTabBar)
- **之前**: `font-weight: bold`
- **现在**: `font-weight: bold` (保持)
- **效果**: 标签文字已经很粗

#### 分组框标题 (QGroupBox::title)
- **之前**: `font-weight: bold`
- **现在**: `font-weight: bold` (保持)
- **效果**: 标题已经很粗

#### 状态栏 (QStatusBar)
- **之前**: 默认字重
- **现在**: `font-weight: 600`
- **效果**: 状态信息更清晰

---

### 2. 圆角美化优化

#### 主窗口 (QMainWindow)
- **之前**: 无圆角
- **现在**: `border-radius: 12px`
- **效果**: 窗口边缘更柔和

#### 分组框 (QGroupBox)
- **之前**: `border-radius: 10px`, `border: 1px`
- **现在**: `border-radius: 15px`, `border: 2px`
- **效果**: 更大的圆角，更粗的边框，更突出

#### 分组框标题
- **之前**: 无背景，无圆角
- **现在**: 带背景色 `#2C2F33`，`border-radius: 8px`
- **效果**: 标题像一个独立的标签，更美观

#### 输入框/文本框 (QLineEdit, QTextEdit, QComboBox)
- **之前**: `border-radius: 5px`, `border: 1px`, `padding: 6px`
- **现在**: `border-radius: 8px`, `border: 2px`, `padding: 8px`
- **效果**: 更大的圆角，更粗的边框，更宽松的内边距

#### 按钮 (QPushButton)
- **之前**: `border-radius: 5px`, `padding: 8px 16px`
- **现在**: `border-radius: 8px`, `padding: 10px 18px`
- **效果**: 更大的圆角，更宽松的内边距

#### 标签页窗格 (QTabWidget::pane)
- **之前**: `border-radius: 5px`, `border-top: 1px`
- **现在**: `border-radius: 8px`, `border-top: 2px`
- **效果**: 更大的圆角，更粗的顶部边框

#### 标签页标签 (QTabBar::tab)
- **之前**: `border-radius: 8px`, `padding: 10px 25px`, `border: 1px`, `margin-right: 2px`
- **现在**: `border-radius: 10px`, `padding: 12px 28px`, `border: 2px`, `margin-right: 3px`
- **效果**: 更大的圆角，更宽松的内边距，更粗的边框，更大的间距

#### 滚动条滑块 (QScrollBar::handle)
- **之前**: `border-radius: 5px`, `min-height/width: 20px`
- **现在**: `border-radius: 6px`, `min-height/width: 25px`
- **效果**: 稍大的圆角，更长的滑块

#### 复选框 (QCheckBox::indicator)
- **之前**: `border-radius: 3px`, `width/height: 15px`, `border: 1px`
- **现在**: `border-radius: 5px`, `width/height: 18px`, `border: 2px`
- **效果**: 更大的圆角，更大的尺寸，更粗的边框

#### 工具按钮 (QToolButton)
- **新增**: `border-radius: 6px`, `border: 2px`, `padding: 6px`
- **效果**: 刷新按钮等工具按钮有圆角和边框

#### 下拉框箭头区域 (QComboBox::drop-down)
- **新增**: `border-radius: 6px` (仅右侧圆角)
- **效果**: 下拉箭头区域与输入框整体协调

## 🎯 视觉效果改进

### 整体感受
- ✨ **更现代**: 大圆角带来苹果式的现代设计感
- ✨ **更清晰**: 加粗字体提升可读性，特别是在低DPI显示器上
- ✨ **更醒目**: 更粗的边框让组件边界更清晰
- ✨ **更宽松**: 增加的内边距让界面呼吸感更好

### 具体改进点

#### 1. 分组框
- 边框从1px增加到2px，更明显
- 圆角从10px增加到15px，更柔和
- 标题增加背景和圆角，像一个浮动标签
- 增加内边距15px，内容不贴边

#### 2. 输入控件
- 所有输入框边框加粗到2px
- 圆角统一增加到8px
- 内边距从6px增加到8px
- 聚焦时边框颜色更明显

#### 3. 按钮
- 圆角从5px增加到8px
- 内边距增加，按钮看起来更饱满
- 按下时的位移效果更明显 (12px vs 10px)

#### 4. 标签页
- 圆角从8px增加到10px
- 标签宽度增加（padding: 12px 28px）
- 边框加粗到2px
- 标签间距增加到3px

#### 5. 复选框
- 尺寸从15px增加到18px，更容易点击
- 圆角从3px增加到5px
- 边框加粗到2px

## 📱 响应式设计

所有圆角和边框的调整都保持了比例协调：

```
超小圆角 (3px → 5px): 复选框
小圆角 (5px → 6-8px): 输入框、按钮、工具按钮
中圆角 (8px → 10px): 标签页标题
大圆角 (10px → 12-15px): 分组框、主窗口
```

## 🚀 使用建议

### 最佳显示效果
- 显示器分辨率: 1920x1080 或更高
- 系统DPI缩放: 100% 或 125%
- 操作系统: Ubuntu 22.04 / Windows 10+ / macOS

### 如果觉得字体太粗
可以在代码中调整全局字体权重：

```python
# 找到第35行
font-weight: 600;

# 改为
font-weight: 500;  # 稍细
```

### 如果觉得圆角太大
可以统一减小圆角值，建议按比例调整：

```python
# 主窗口: 12px → 8px
# 分组框: 15px → 10px
# 输入框/按钮: 8px → 6px
# 标签页: 10px → 8px
```
